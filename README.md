# OLJ Web Scraper (React + TypeScript)

A minimal React + TypeScript web UI for a web scraper, built with Vite.  
It includes a keyword search box, a search button, and a results list placeholder.

## Getting started

1. Install dependencies:

```bash
cd "d:\Development\OLJ scrapper"
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open the URL shown in the terminal (by default `http://localhost:5173`) in your browser.

## Next steps

- Replace the placeholder search logic in `src/App.tsx` with a real API call to your backend scraper.
- On the backend, perform the actual web scraping and return JSON results that match the `ScrapeResult` shape in `App.tsx`.
